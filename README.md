# RUCHE

RUCHE (short for ROS2-based Unified Control for Hugging-face Embodied-agents) is [B-AROL-O Team](https://github.com/B-AROL-O) response to "[MCP'st 1st Birthday](https://huggingface.co/MCP-1st-Birthday)" Hackathon hosted by Anthropic and Gradio with additional support from Hugging Face, OpenAI, Gemini, Modal, Sambanova, IIElevenLabs, Blaxel, Llamaindex and Nebius.
